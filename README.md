# Idaproject Nuxt 3

## Проект собран на Nuxt 3 + Typescript

Ссылка на демо - [Idaproject vue3 demo](https://)

## Версии библиотек
- NUXT - 3.0.0-rc.6
- NPM - 8.16 
- Node - 16.16 

## Запуск проекта

### Загрузить все зависимости проекта:  
- `npm install`

### Запуск в режиме разработки:  
- `npm run dev`

#### Запуск режима просмотра:  
- `npm run preview`

Запуск для деплоя:  
- `npm run lint`  
- `npm run build`
